# История изменений

Все заметные изменения этого проекта документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
проект использует [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Невыпущено]

## [0.3.1] - 2026-06-17

### Изменено

- Исключены из Git сгенерированные каталоги `.claude-plugin/` и `build/`.
- Уточнены русские формулировки в навыках без изменения маршрутов и поведения.
- Проверки срабатывания выполняются по навыкам, а сценарии результата — по
  одному, чтобы тяжёлые проверки не превышали тайм-аут из-за объединённого
  контекста.
- Уточнён маршрутизатор `ai-skill-development` для задач про подключение общих
  правил к проектным навыкам.
- Уточнён маршрутизатор `ai-data-collection` для сбора данных перед изменением
  существующих источников истины.
- Уточнён маршрутизатор `ai-apm-setup` для вопросов о необходимости
  `apm compile` в тестовом цикле коллекции навыков.
- Усилен самоконтроль `ai-agents-md-maintenance` после разделения корневых и
  вложенных инструкций.

### Исправлено

- Модельные проверки навыков изолированы от пользовательского `config.toml`.
- Если модель не возвращает результат проверки срабатывания, сценарий
  перезапускается отдельно; неверные классификации по-прежнему считаются
  ошибкой.
- Проверяемая модель теперь получает `expected_output`, `assertions` и
  `must_not`, чтобы сценарии результата проверяли применение навыка, а не
  угадывание скрытых ожиданий.
