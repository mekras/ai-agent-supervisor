# Журнал решений

В этом каталоге хранятся записи значимых решений проекта. Новую запись создают,
когда решение задаёт устойчивое правило, имеет долгие последствия или требует
заметных затрат на отмену.

## Правила журнала

- Все записи используют один журнал без разделения по видам решений.
- Имя файла: `DR-NNNN-краткое-название.md`, где `NNNN` — четырёхзначный номер.
- В записи указывают контекст, рассмотренные варианты, принятое решение,
  последствия и способ проверки.
- Статус записи: «Предложено», «Принято», «Отклонено», «Устарело» или
  «Заменено».
