# ADR-0001: Приоритеты развития коллекции навыков

**Статус:** принято 2026-07-10

## Контекст

Коллекция `.apm/skills/` развивается для использования в других проектах.
Изменения могут повышать качество, увеличивать стоимость модельного контекста,
упрощать применение или расширять совместимость, поэтому нужен устойчивый
порядок разрешения компромиссов.

## Решение

Альтернативы оцениваются в таком порядке:

1. качество результатов, обеспечиваемых навыками;
2. стоимость для конечного пользователя, включая модельные и контекстные
   накладные расходы;
3. простота применения;
4. спектр поддерживаемых моделей и оснасток.

К следующему критерию переходят только если альтернативы сопоставимы по
предыдущему.

## Последствия

- Новый материал не добавляется в навык только ради охвата, если он не улучшает
  проверяемый результат.
- Постоянно загружаемый контекст и дополнительные вызовы оцениваются как
  пользовательская стоимость.
- Совместимость не оправдывает ухудшение качества либо необоснованный рост
  затрат.
