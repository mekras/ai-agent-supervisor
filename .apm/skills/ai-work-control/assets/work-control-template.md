# Шаблон контроля применения ИИ

## Быстрый режим

```text
Задача:
Ожидаемый результат:
Достаточный контекст:
Недостающий контекст:
Главный риск:
Граница самостоятельности:
Следующий маршрут:
Решение: применять | пауза | переопределить задачу | полный контроль | отдать человеку
```

## Полный режим

```text
Задача:
Ожидаемый результат:
Почему задача подходит для ИИ:
Текущий контекст:
Использованные источники:
Недостающие данные:
Применяемые правила проекта:
Нужные роли, навыки и процедуры:
Маршрут:
Что ИИ делает самостоятельно:
Что ИИ готовит только как предложение:
Что требует решения человека:
Основные риски:
Как снижаются риски:
Как проверяется результат:
Нужна ли ai-work-result-evaluation:
Что нельзя считать завершённым без решения человека:
Решение по запуску:
```

## Правила заполнения

- Отделяй факты, допущения, выводы и предложения.
- Не скрывай недостающий контекст внутри общего описания.
- Не переходи к проектированию источника истины без полного режима.
- При переходе из быстрого режима в полный переноси уже установленные сведения
  и дополняй только недостающие поля.
- Если способ проверки результата не определён, задача не готова к выполнению.
