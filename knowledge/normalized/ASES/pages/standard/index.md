# Autonomous Systems Evaluation Standard (нормализованное представление)

## URL

- `https://ukgovernmentbeis.github.io/as-evaluation-standard/`

## Ключевые тезисы

- Стандарт задаёт минимальные требования и рекомендации для подачи новой
  оценки в набор `Autonomous Systems` института `AISI`.
- Обязательная технологическая основа: все оценки должны быть построены на
  `Inspect`.
- Репозиторий оценки должен соответствовать шаблону `as-evaluation-standard`
  и включать обязательные артефакты `README.md`, `CONTRIBUTING.md` и
  `METADATA.json`.
- Оценка должна проходить автоматические проверки и хранить evidence по
  quality assurance, включая хотя бы один `Inspect`-журнал успешного прогона.
- Для каждой оценки требуются собственные тесты, а scoring должен быть
  полностью автоматическим без ручной проверки.
- Документ отдельно ограничивает небезопасные и неэтичные сценарии и
  рекомендует моделировать опасные действия через симуляцию или подзадачи.
- В рекомендациях источник продвигает бинарную основную метрику, дополнительные
  метрики, минимизацию ручной настройки окружения, наблюдаемость траекторий и
  консультацию с предметным экспертом.

## Ограничения нормализации

- Нормализация основана на локальном HTML-снимке из `knowledge/primary/ASES`.
- Полный текст страницы не переносится в Git из-за неустановленного
  лицензионного статуса.
- Детали примеров, ссылки на шаблонные файлы и второстепенные советы сведены к
  краткому пересказу.
