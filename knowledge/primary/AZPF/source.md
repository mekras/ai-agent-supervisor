# Источник: `AZPF`

## Статус

Источник принят в корпус знаний как внешний публичный веб-материал
с ограничением на хранение полного HTML в Git.

## Происхождение

- Сайт: `https://learn.microsoft.com`
- URL:
  `https://learn.microsoft.com/en-us/azure/machine-learning/prompt-flow/how-to-develop-an-evaluation-flow`
- Дата получения: `2026-06-10T07:41:37Z`
- Способ получения: HTTP GET с локальным сохранением raw HTML и заголовков
  ответа.
- Получена 1 страница с HTTP-статусом `200`.

## Что сохранено

В состав источника включены следующие артефакты:

- passport: `knowledge/primary/AZPF/source.md`
- индекс: `knowledge/primary/AZPF/page-index.tsv`
- страница: `knowledge/primary/AZPF/pages/evaluation-flow/index.html`
- заголовки ответа:
  `knowledge/primary/AZPF/pages/evaluation-flow/response-headers.txt`

Снимок хранится локально с исключением `pages/` из Git через
`knowledge/primary/AZPF/.gitignore`.

## Ограничения

- Источник ограничен указанной страницей Microsoft Learn.
- Снимок отражает состояние страницы на момент получения.
- Полный текст страницы не должен использоваться как единственная опора без
  проверки условий использования.

## Условия использования

- На уровне сохранённого HTML явное публичное разрешение на переиспользование
  полного текста не зафиксировано.
- Уровень уверенности: средний.
- Хранение полного HTML-артефакта в Git публичного проекта не допускается без
  отдельного подтверждения условий.
- Для внутренних задач допустимо извлечение проверяемых утверждений на основе
  сохранённого снимка.
- Атрибуция и правовой статус: `knowledge/source-attribution.md`.

## Очистка перед сохранением

В заголовках ответа удалены значения `set-cookie` и служебная строка прокси
`HTTP/1.1 200 Connection established`, если они присутствовали.

## Назначение в проекте

Материал может использоваться как первичный для изучения evaluation flow в
Azure Machine Learning prompt flow.
