# Источник: `APMP`

## Статус

Источник принят в корпус знаний как внешний публичный веб-материал
с ограничением на хранение полного HTML в Git.

## Происхождение

- Платформа: `https://microsoft.github.io/apm`
- URL:
  `https://microsoft.github.io/apm/producer/` и дочерние страницы внутри этого
  префикса.
- Дата пакетного получения: `2026-06-08T13:15:55Z` — `2026-06-08T13:16:16Z`
- Способ получения: HTTP GET с локальным сохранением raw HTML и заголовков
  ответа.
- Получено 12 страниц с HTTP-статусом `200`.
- `Last-Modified` по ответу сервера указан отдельно для каждой страницы в
  индексе; для текущего набора страниц значение одинаковое:
  `Thu, 04 Jun 2026 15:35:43 GMT`.

## Что сохранено

В состав источника включены следующие артефакты:

- passport: `knowledge/primary/APMP/source.md`
- индекс: `knowledge/primary/APMP/page-index.tsv`
- страницы (сырые HTML-снимки):
  - `knowledge/primary/APMP/pages/**/index.html`
- заголовки ответа:
  - `knowledge/primary/APMP/pages/**/response-headers.txt`

Снимок хранится локально с исключением `pages/` из Git через
`knowledge/primary/APMP/.gitignore`.

## Ограничения

- Источник зафиксирован как набор страниц раздела producer в документации APM.
- В охват входят только страницы внутри префикса
  `https://microsoft.github.io/apm/producer/`.
- Снимок отражает состояние раздела на момент получения.
- Полный текст страниц не должен использоваться как единственная опора без
  проверки условий использования.

## Условия использования

- На уровне сохранённого HTML явное публичное разрешение на переиспользование
  полного текста не зафиксировано.
- Уровень уверенности: средний.
- Хранение полного HTML-артефакта в Git публичного проекта не допускается без
  отдельного подтверждения условий.
- Для внутренних задач допустимо извлечение проверяемых утверждений на основе
  сохранённого снимка.
- Атрибуция и правовой статус: `knowledge/source-attribution.md`.

## Очистка перед сохранением

В заголовках ответа удалены значения `set-cookie` и служебная строка прокси
`HTTP/1.1 200 Connection established`, если они присутствовали.

## Назначение в проекте

Материал может использоваться как первичный для изучения:

- producer-процесса APM и структуры пакета;
- правил публикации и упаковки навыков;
- совместимости локальных skills с каналом установки через APM;
- различий между переносимыми и target-specific primitives.
