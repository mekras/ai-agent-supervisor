`APMP`

- Состояние пригодности: пригоден для дальнейшей работы с оговорками

## Что это за источник

Внешний публичный веб-источник: документация `Agent Package Manager` на
`microsoft.github.io`, разделы `producer` и `consumer`.

Сейчас в корпус включены 14 страниц с HTTP-статусом `200`:

- 12 страниц внутри префикса `https://microsoft.github.io/apm/producer/`,
  включая обзорную страницу раздела и дочерние страницы по authoring, compile,
  preview, pack и publish;
- 2 страницы раздела `https://microsoft.github.io/apm/consumer/`:
  `manage-dependencies` и `drift-and-secure-by-default`. Они описывают работу
  проекта-потребителя с зависимостями: манифест, файл блокировки, проверки
  установки и команду `apm audit`.

## Почему источник относится к проекту

Источник рассматривается как первичный материал для изучения:

- структуры и публикации пакетов APM;
- оформления skills, prompts, instructions, agents, hooks, commands и MCP;
- сборки, проверки и публикации пакетов для сред, где этот репозиторий
  используется через APM;
- работы проекта-потребителя с зависимостями APM: различия `apm.yml` и
  `apm.lock.yaml`, прямых и транзитивных зависимостей, проверок установки и
  команды `apm audit`.

## Условия использования

- Явная лицензия на переиспользование полного HTML не подтверждена.
- Уровень уверенности: средний.
- Полный HTML-контент в Git публичного репозитория не хранится; допускается
  внутренний анализ по локальным снимкам с атрибуцией в производных.
- Нормализованные текстовые представления страниц не должны публиковаться в
  открытом Git без отдельного правового основания.
