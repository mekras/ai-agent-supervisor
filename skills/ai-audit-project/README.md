# Навык проверки агентских правил проекта.

Навык помогает проверять набор правил проекта для агента: `AGENTS.md`, файлы
инструкций разных инструментов, локальные навыки и связанные справки.

## Применение

Попросите агента проверить правила проекта, например, такими запросами:

```text
Проверь агентские правила проекта.
```
```text
Проверь все правила агента в проекте.
```

Для Codex можно применить навык по имени:

```text
$ai-audit-project
```
